# V2.2 — Listas Independentes e Histórico Estruturado

Data: 2026-07-15

## Objetivo

Separar definitivamente perfis de coordenadas e listas programadas, permitindo reutilizar qualquer lista com qualquer perfil compatível. Organizar o histórico por sessão.

## Arquitetura

```txt
PERFIS DE COORDENADAS
  independentes

LISTAS PROGRAMADAS
  nome
  data
  descrição
  sinais
  status calculado

SESSÃO
  perfil + lista

HISTÓRICO
  sessões + eventos
```

## Status das listas

```txt
HOJE
AMANHÃ
FUTURA
HISTÓRICO
```

O status é calculado pela data atual. Listas antigas ou arquivadas são classificadas como Histórico.

## Cadastro de lista

Campos:

- nome;
- dia;
- mês;
- ano;
- descrição opcional.

Após salvar, a aplicação abre a Agenda para inclusão dos horários.

## Histórico estruturado

Cada sessão registra:

- início e fim;
- perfil;
- lista e data;
- status final;
- eventos;
- cliques enviados;
- sinais expirados;
- cancelamentos;
- erros.

A tela permite filtrar, limpar com confirmação e exportar CSV.

## Migração

O formato de configuração passa para a versão 4. Perfis e listas da V2.1 são migrados automaticamente. As listas deixam de permanecer dentro dos perfis.
