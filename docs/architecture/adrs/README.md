## Lifecycle canônico

O template oficial é [`ADR_TEMPLATE.md`](ADR_TEMPLATE.md) e define:

```text
PROPOSED_FOR_REVIEW
ACCEPTED
SUPERSEDED
DEPRECATED
REJECTED
```

`ACCEPTED` significa decisão arquitetural aprovada e publicada. Não significa implementação autorizada, runtime validado ou Modo LIVE armado.
